## 2. 浏览器运行验收（手动）

- [ ] **加载**：双击 index.html 或启动 dev-server，无控制台报错/警告
- [ ] **加载层**：400ms 后淡出，显示「正在构建世界…」
- [ ] **画布**：全屏渲染，窗口缩放无变形
- [ ] **FPS**：初始 FPS ≥ 30（右下角 HUD）

## 3. 场景渲染验收（手动）

- [ ] **天空**：着色器渐变球无接缝，颜色随时刻平滑过渡
- [ ] **太阳**：太阳精灵位置与方向光一致，随时刻移动
- [ ] **星空**：夜晚（< 7:00 / > 19:00）显示星空 Points
- [ ] **阴影**：建筑在地面投射软阴影（PCFSoft），无 acne
- [ ] **色调**：ACES 色调映射生效，画面无过曝
- [ ] **地形**：起伏自然，无棋盘格规律，中心下沉海盆
- [ ] **水面**：半透明，可见水下地形，有轻微波动，无 Z-fighting
- [ ] **建筑**：CBD 高外围低，窗户纹理 UV 正确不拉伸
- [ ] **窗户发光**：夜晚部分窗户 emissive 点亮，白天熄灭
- [ ] **植被**：树木投阴影，树冠 flatShading，草丛不卡顿
- [ ] **雾**：随时刻变色，浓度滑块实时生效

## 4. 交互验收（手动）

- [ ] **时刻滑块**：天空/太阳/雾/窗户实时联动
- [ ] **日出/日落**：暖色调过渡（7点/17点）
- [ ] **密度滑块**：三档可切换，重建后建筑数量变化
- [ ] **相机三模式**：环绕/飞行/漫游切换，按钮高亮，无相机跳变
- [ ] **轨道模式**：拖拽旋转 + 阻尼，极角不翻入地下
- [ ] **飞行模式**：平滑曲线绕飞，不穿地
- [ ] **漫游模式**：WASD 移动顺滑，Esc 退出指针锁定
- [ ] **重置视角**：相机回到初始位置
- [ ] **重建世界**：完全重建，加载层闪烁

## 5. 移动端验收（手动）

- [ ] **触摸**：单指拖拽旋转，双指缩放
- [ ] **面板**：≤ 768px 宽度面板可折叠/隐藏
- [ ] **viewport**：无 300ms 延迟，meta 正确
- [ ] **横竖屏**：切换时自适应

## 6. 性能验收（Chrome DevTools）

- [ ] **FPS**：稳态 ≥ 30（中端设备），目标 60
- [ ] **三角面**：HUD 显示数值，DevTools 一致
- [ ] **Draw Calls**：≤ 300
- [ ] **内存**：重建后无持续增长（Heap Snapshot 对比）
- [ ] **首屏**：加载 < 2s（CDN 缓存命中）
