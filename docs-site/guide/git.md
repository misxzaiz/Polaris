# Git 版本控制

Polaris 内置 Git 面板，提供完整的版本控制操作。

通过 ActivityBar 的 **GitPullRequest** 图标打开 Git 面板。

## 标签页

Git 面板包含 7 个标签页：

| 标签 | 说明 |
|------|------|
| 变更 | 查看和提交文件变更 |
| 历史 | 浏览提交历史 |
| 分支 | 管理本地和远程分支 |
| 远程 | 管理远程仓库（添加/删除/推送/拉取） |
| 标签 | 创建和管理 Git 标签（附注/轻量） |
| Stash | 暂存和恢复工作区更改 |
| .gitignore | 管理忽略规则（支持模板） |

## 变更 (Changes)

### 文件状态

| 状态 | 说明 |
|------|------|
| 已暂存 | 已添加到暂存区，将包含在下一次提交中 |
| 未暂存 | 已修改但未暂存 |
| 未跟踪 | 新文件，尚未被 Git 追踪 |
| 没有变更 | 工作区干净 |

### 变更类型

- **新增 (A)** — 新文件
- **修改 (M)** — 已修改文件
- **删除 (D)** — 已删除文件
- **重命名 (R)** — 重命名文件

### 操作

| 操作 | 说明 |
|------|------|
| 暂存全部 | 将所有未暂存变更添加到暂存区 |
| 取消暂存全部 | 将所有已暂存变更移回未暂存 |
| 丢弃全部 | 撤销所有未暂存的修改（不可恢复） |
| 全选 | 批量选择文件 |
| AI 生成提交信息 | 让 AI 根据变更自动生成提交描述 |

### 提交

在底部输入框填写提交信息，点击「提交」按钮。支持 AI 自动生成提交信息。

### 差异查看

点击变更文件查看 diff 视图，显示行级增删，支持在 CenterStage 中打开。

## 分支 (Branch)

| 功能 | 说明 |
|------|------|
| 当前分支 | 显示当前所在分支 |
| 本地分支 | 列出所有本地分支，可切换 |
| 远程分支 | 列出所有远程分支 |
| 创建分支 | 从当前分支创建新分支 |
| 删除分支 | 删除指定分支（需确认） |
| 重命名分支 | 重命名本地分支 |
| 合并分支 | 将指定分支合并到当前分支，支持禁用快进合并 |
| 变基分支 | 将当前分支变基到指定分支 |
| 设置上游 | 为推送设置上游分支 |

合并/变基遇到冲突时显示冲突文件数，可选择中断或继续操作。

## 远程 (Remote)

| 操作 | 说明 |
|------|------|
| 添加远程仓库 | 输入名称和 URL |
| 拉取 | 从远程仓库拉取更新 |
| 推送 | 推送本地提交到远程 |
| 在浏览器中打开 | 打开远程仓库网页 |
| 复制 URL | 复制远程仓库地址 |
| 删除 | 移除远程仓库配置 |

## 标签 (Tags)

| 类型 | 说明 |
|------|------|
| 附注标签 | 包含创建者信息、日期和消息 |
| 轻量标签 | 仅是一个指向特定提交的指针 |

填写消息将创建附注标签，否则创建轻量标签。

## Stash

| 操作 | 说明 |
|------|------|
| 保存到 Stash | 暂存当前工作区更改，可选包含未跟踪文件 |
| 应用 Stash | 恢复之前暂存的更改 |

## .gitignore 管理

- 从模板快速创建 .gitignore（如 Node.js、Python 等项目模板）
- 快速添加规则（如 `node_modules/`）
- 在线编辑和保存

## 高级操作

| 功能 | 说明 |
|------|------|
| Cherry-pick | 将指定提交应用到当前分支 |
| Revert | 创建新提交来撤销指定提交 |
| Blame | 查看文件每行的最后修改者和提交信息 |

## AI Git 上下文

可在 AI 对话中使用 Git 上下文：

- **Diff** — 当前变更差异
- **Commit** — 最近提交
- **Log** — 提交历史
- **Branch** — 当前分支状态
- **Status** — 工作区状态
