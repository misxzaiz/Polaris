# 定时任务

定时任务模块支持自动执行 AI 操作，包含完整的任务生命周期管理。

通过 ActivityBar 的 **Clock** 图标打开定时任务面板。

## 调度器控制

面板顶部显示全局调度器状态：

| 状态 | 说明 |
|------|------|
| 调度器运行中 | 正常运行，任务按计划触发 |
| 调度器已停止 | 调度器暂停，需手动启动 |
| 其他实例正在运行 | 另一个应用实例占用了调度器 |

可手动启动/停止调度器，或尝试获取控制权。

## 任务触发类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 单次执行 | 指定时间执行一次 | 2026-04-12 18:00 |
| Cron 表达式 | 使用 Cron 语法灵活调度 | `0 9 * * 1-5`（工作日早9点） |
| 间隔执行 | 按固定时间间隔循环 | 每 30 分钟 |
| 完成后间隔 | 上次执行完成后再等待指定时间 | 完成后等 10 分钟 |

## 任务模式

| 模式 | 说明 |
|------|------|
| 简单模式 | 直接配置提示词和触发条件 |
| 协议模式 | 使用协议模板，支持参数化配置和任务目标 |

## 任务分类

内置分类：开发任务、审查任务、新闻搜索、监控任务、自定义。

## 创建任务

### 基本配置

| 字段 | 必填 | 说明 |
|------|------|------|
| 任务名称 | 是 | 例如「每日日报生成」 |
| 描述 | 否 | 任务的补充说明 |
| 任务模式 | 是 | 简单模式 / 协议模式 |
| 任务分类 | 是 | 选择内置分类或自定义 |
| 触发配置 | 是 | 触发类型和触发值 |
| AI 引擎 | 是 | 选择模型提供商和模型 |
| 提示词 | 是 | AI 要执行的内容 |
| 工作目录 | 否 | 留空使用默认目录 |

### 高级配置

| 字段 | 说明 |
|------|------|
| 分组 | 任务的分组标签 |
| 最大执行次数 | 限制任务执行总次数 |
| 最大重试次数 | 失败后自动重试 |
| 重试间隔 | 重试之间的等待时间 |
| 超时时间 | 单次执行的最大时长（分钟） |

### 协议模式

协议模式使用预定义的协议模板，包含：

| 配置 | 说明 |
|------|------|
| 任务目标 | 描述任务的主要目标，AI 持续执行直到完成 |
| 参数 | 可定义文本、多行文本、下拉选择、数字、日期等类型参数 |
| 执行规则 | 指导 AI 如何执行任务 |
| 记忆规则 | AI 在多次执行间的记忆策略 |

### 提示词模板

可创建可复用的提示词模板，支持变量插值：

| 变量 | 说明 |
|------|------|
| `{{prompt}}` | 用户实际输入的提示词 |
| `{{taskName}}` | 任务名称 |
| `{{date}}` | 当前日期 |
| `{{time}}` | 当前时间 |
| `{{datetime}}` | 当前日期时间 |
| `{{weekday}}` | 当前星期几 |

## 任务管理

### 列表操作

| 操作 | 说明 |
|------|------|
| 立即执行 | 不等待定时，立即运行一次 |
| 查看日志 | 查看历史执行记录 |
| 编辑 | 修改任务配置 |
| 复制 | 基于当前任务创建副本 |
| 删除 | 删除任务 |
| 启用/禁用 | 切换任务激活状态 |

### 筛选和排序

支持按状态（已启用/已禁用）、触发类型、AI 引擎筛选。排序支持创建时间、更新时间、任务名称、上次/下次执行时间。

## 执行日志

每次任务执行生成日志记录：

- 实时流式输出（支持自动/手动滚动）
- 显示执行用时
- 记录执行状态：等待中 → 执行中 → 成功/失败
- 可清空日志
- 显示下次执行倒计时
