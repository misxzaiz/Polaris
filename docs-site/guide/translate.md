# 翻译

Polaris 内置翻译工具，支持中英互译。基于百度翻译 API。

通过 ActivityBar 的 **Languages** 图标打开翻译面板。

## 使用方式

### 基本翻译

1. 选择翻译方向：**中 → 英** 或 **英 → 中**
2. 在「原文」区域输入待翻译文本
3. 点击「翻译」按钮，译文实时显示

### 翻译后操作

| 操作 | 说明 |
|------|------|
| 发送到对话 | 将译文直接发送到右侧 AI 对话面板 |
| 发送到面板 | 将译文填入对话输入框 |
| 复制引用 | 复制翻译结果 |
| 引用问 AI | 以翻译结果为上下文向 AI 提问 |

### AI 问答

翻译后可基于引用内容向 AI 提问：

1. 翻译完成后点击「引用问 AI」
2. 在「你的问题」输入框中输入问题
3. AI 会结合翻译原文回答

## 历史记录

翻译结果自动保存到历史记录，支持搜索。

## 配置

翻译功能需要配置百度翻译 API：

1. 进入 **设置 → 翻译**
2. 填入 **App ID** 和 **密钥 (Secret Key)**
3. 前往[百度翻译开放平台](https://fanyi-api.baidu.com/)申请

## 错误码

| 错误码 | 含义 |
|--------|------|
| 52000 | 成功 |
| 52001 | 请求超时 |
| 52002 | 系统错误 |
| 52003 | 未授权用户 |
| 54000 | 必填参数为空 |
| 54001 | 签名错误 |
| 54003 | 访问频率受限 |
| 58000 | 客户端IP非法 |
| 58001 | 译文语言方向不支持 |
| 58002 | 服务当前已关闭 |
