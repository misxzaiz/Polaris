# 工作区

工作区是 Polaris 管理项目和配置的基本单元。每个工作区绑定一个本地目录。

## 创建工作区

通过顶栏的工作区选择器创建新工作区：

| 字段 | 说明 |
|------|------|
| 工作区名称 | 例如「我的项目」 |
| 工作区路径 | 选择项目文件夹（可点击「浏览」按钮选取） |
| 切换到新工作区 | 开启则创建后立即切换；关闭则仅添加到关联工作区 |

## 工作区信息面板

展开工作区选择器可查看：

- 当前工作区名称和路径
- 关联工作区列表
- 文件引用语法提示

## 关联工作区

关联工作区允许 AI 访问多个项目目录的文件：

- 引用当前工作区文件：`@/path/to/file`
- 引用关联工作区文件：`@{工作区名}:path/to/file`
- 可动态添加或移除关联工作区
- AI 可访问所有关联工作区中的文件

## 创建新会话

新建会话时可选择：

| 配置 | 说明 |
|------|------|
| 主工作区 | 会话绑定的主要工作区 |
| 关联工作区 | AI 可额外访问的工作区（可选） |

## 切换和删除

- 通过顶栏工作区选择器快速切换
- 删除工作区需确认（不可撤销）
- 工作区数据独立维护（会话历史、System Prompt、面板布局）

## 数据存储

- 会话历史持久化到本地存储
- 支持跨会话恢复上下文
- 空闲会话自动释放内存，切换时从快照恢复
- 最多保留 20 个内存快照
