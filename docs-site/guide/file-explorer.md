# 文件管理

Polaris 内置文件浏览器和代码编辑器，无需离开应用即可管理项目文件。

## 文件浏览器

通过 ActivityBar 的 **Files** 图标打开左侧文件浏览器。

### 工具栏

| 操作 | 说明 |
|------|------|
| 新建文件 | 在当前目录创建新文件 |
| 新建文件夹 | 在当前目录创建新文件夹 |
| 全部折叠 | 折叠所有展开的目录 |
| 刷新 | 重新加载文件树 |
| 显示/隐藏隐藏文件 | 切换隐藏文件的显示 |

### 搜索

在搜索框中输入关键词可模糊搜索文件名，实时显示匹配数量。

### 右键菜单

| 操作 | 说明 |
|------|------|
| 新建文件 / 新建文件夹 | 在当前目录下创建 |
| 剪切 / 复制 / 粘贴 | 文件操作 |
| 重命名 | 修改文件或文件夹名称 |
| 删除 | 删除文件或文件夹（含确认弹窗） |
| 打开文件 | 在编辑器中打开 |
| 打开文件夹 | 展开/折叠文件夹 |
| 在浏览器中打开 | 使用系统默认程序打开 |
| 在资源管理器中打开 | 打开系统文件管理器并定位 |
| 复制路径 | 复制文件的完整路径到剪贴板 |

## 代码编辑器

双击文件在 CenterStage 区域打开编辑器。

### 标签页

- 同时打开多个文件，标签页切换
- 未保存的文件在标签上显示黄色圆点标记
- 右键标签页支持：关闭、关闭其他、关闭所有、关闭右侧、关闭已保存
- 可复制文件路径或相对路径
- 在文件浏览器中定位文件

### 编辑器功能

| 功能 | 说明 |
|------|------|
| 语法高亮 | 支持 JavaScript, TypeScript, Python, Rust, Go, Java, C++, HTML, CSS, JSON, XML, SQL, Markdown, YAML 等 20+ 语言 |
| 代码折叠 | 折叠/展开代码块 |
| 搜索替换 | 文件内查找和替换 |
| 差异对比 | Git 变更的 diff 视图，显示行添加/删除 |
| 图片预览 | 支持直接预览图片文件 |

### 未保存更改

关闭未保存的文件时会弹出确认对话框，可选择：

- **取消** — 取消关闭操作
- **放弃更改** — 不保存直接关闭
- **保存并关闭** — 保存后关闭

## 文件路径

文件路径显示在编辑区上方的面包屑导航中。应用使用 `@/path` 语法引用当前工作区文件，使用 `@{name}:path` 引用关联工作区文件。
