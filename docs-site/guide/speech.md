# 语音

Polaris 支持语音输入和语音输出（TTS），在 **设置 → 语音输入** 中配置。

## 语音输入

| 配置 | 说明 |
|------|------|
| 启用语音输入 | 开启后，在状态栏点击语音按钮开始连续语音识别 |
| 识别语言 | 选择语音识别的目标语言 |

### 语音命令

开启语音输入后，可使用语音命令控制应用：

| 命令 | 功能 |
|------|------|
| 「发送消息」 | 发送当前输入框中的内容 |
| 「清空输入框」 | 清空输入框 |
| 「中断对话」 | 中断 AI 正在生成的回复 |
| 「撤回最后输入」 | 撤回最后一条用户消息 |

### 使用提示

- 点击状态栏语音按钮开始/停止识别
- 开启后持续识别，直到手动关闭
- 需要麦克风权限和网络连接

## 语音输出 (TTS)

| 配置 | 说明 |
|------|------|
| 启用语音输出 | AI 回复完成后自动朗读文本内容 |
| 语音角色 | 选择 TTS 语音角色 |
| 语速 | 调整朗读语速 |
| 音量 | 调整朗读音量 |
| 自动播放 | 消息完成后自动朗读；关闭后需手动触发 |

### 测试

可在设置页面点击「测试语音」按钮试听效果。

### 行为说明

- 自动过滤代码块、工具调用等内容，只朗读纯文本
- 发送新消息或中断对话时会自动停止当前朗读
- 语音合成需要网络连接
