# Polaris Web 全流程测试方案

> 测试目标: `http://192.168.43.244:9800`
> 测试环境: 浏览器 (Chrome) 通过 HTTP + WebSocket 访问
> 认证方式: Token 认证 (`Authorization: Bearer {token}`)

## 八、Bug 分级标准

| 等级 | 定义 | 示例 |
|------|------|------|
| P0 - 致命 | 功能完全不可用，白屏/崩溃 | 页面无法加载、认证失败 |
| P1 - 严重 | 核心功能异常 | Chat 无法发送、文件无法打开 |
| P2 - 一般 | 功能可用但体验异常 | 布局错位、样式问题 |
| P3 - 轻微 | UI 文案/细节问题 | 按钮文字截断、间距不一致 |
