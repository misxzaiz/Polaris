优化要点

  • 移除卡片嵌套：阶段不再使用独立边框卡片，改用左侧竖线连接
  • 扁平化层次：主卡片 → 分隔线 → 阶段列表 → 任务，清晰的视觉层次
  • 紧凑间距：padding 从 12px 优化为 8-10px，整体高度降低约 30%
  • 简化图标：状态用小圆点表示，阶段用带数字/符号的圆圈
  • 内联任务：任务直接显示在阶段下方，无需展开容器
  • 精简按钮：移除图标，仅保留文字，减少视觉噪音
