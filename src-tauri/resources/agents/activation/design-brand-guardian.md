交付物：

2. 视觉识别系统（颜色用 CSS 变量、字体、间距）
3. 品牌语调和信息架构
4. 品牌使用规范
5. [如果是审计]：品牌一致性报告，标出具体偏差

要求：
- 所有颜色给 hex 值，方便 CSS 实现
- 字体指定 Google Fonts 或系统字体方案
- 语调规范附"该做"和"不该做"的例子
- 颜色组合满足无障碍对比度要求（WCAG AA）
